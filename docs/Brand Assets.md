# Brand Assets

## Purpose

This document defines all official brand assets used across the Relvio ecosystem.

It ensures consistency in every touchpoint, including the product, website, social media, documentation, presentations, and marketing.

---

# Official Brand Name

**Relvio**

Always capitalize as shown.

Never use:

- relvio
- RELVIO
- RelVio
- Relvio OS

---

# Brand Tagline

Primary

> **Organize People. Build Relationships.**

Alternative

> **The People Operating System.**

---

# Logo Assets

The official logo package includes:

- Primary Logo
- Horizontal Logo
- Vertical Logo
- Icon Only
- Wordmark Only
- Monochrome Logo
- White Logo

---

# App Icon

The app icon should:

- Be recognizable without text
- Use the primary brand color
- Remain clear at small sizes
- Follow platform guidelines

---

# Favicon

Provide:

- 16×16
- 32×32
- 48×48

---

# Logo Clear Space

Maintain clear space around the logo equal to the height of the letter **R** in the wordmark.

No text, graphics, or interface elements should enter this space.

---

# Minimum Logo Size

Digital

```
24px
```

Print

```
15mm
```

The logo should never be scaled below these sizes.

---

Approved versions:

- Full Color
- Black

# Version Control

Every asset update should include:

- Version Number
- Date
- Description
- Owner

Old assets should be archived, not deleted.

---

# Usage Rules

Do:

- Use official assets only.
- Maintain proportions.
- Follow spacing rules.
- Use approved colors.

Don't:

- Recreate the logo.
- Stretch or distort assets.
- Apply effects.
- Modify typography.
- Replace brand colors.

---

# Success Criteria

The brand asset library is successful when:

- Every team uses the same assets.
- Branding remains consistent across all platforms.
- New materials can be created quickly.
- The Relvio identity is instantly recognizable.
