# Design Goals

Every journey should be:

- Completed in as few steps as possible
- Easy for first-time users
- Mobile-friendly
- Fast
- Consistent across the platform

---

# UX Rules

When designing any workflow:

- Avoid unnecessary forms.
- Reduce typing where possible.
- Provide clear feedback after every action.
- Prevent user errors.
- Allow users to undo important actions where appropriate.
