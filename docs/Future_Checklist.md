# Future Features

## Purpose

This document outlines the long-term vision for Atlas beyond the MVP.

These features are not currently planned for the first release but represent opportunities to expand Atlas into the operating system for organizations.

---

# Vision

Atlas should become much more than a people management platform.

It should become the central workspace where organizations manage people, communication, operations, engagement, and growth.

---

# AI Assistant

## Overview

An intelligent assistant that helps organizations save time.

### Capabilities

- Answer natural language questions
- Generate reports
- Summarize attendance
- Suggest follow-ups
- Recommend actions
- Search records using plain English

Example:

> "Show everyone that has missed the last three events."

---

# Workflow Automation

Allow organizations to automate repetitive work.

Examples

- When a visitor attends for the first time, assign a follow-up.
- If someone hasn't attended for 30 days, notify a leader.
- Send reminders before events.
- Automatically move people through journey stages.

---

# Guiding Principle

Future features should only be built if they:

- Solve a meaningful customer problem
- Align with the product vision
- Improve the user experience
- Can be maintained long-term

Avoid adding features simply because competitors have them.

---

# Success Criteria

The long-term success of Atlas is measured by its ability to become the trusted platform organizations rely on daily to manage people, relationships, and growth.
